## 1. 팀 과제 소개
게시판에 학습에 관한 질문을 할 수 있는 학습 게시판, 장비나 코딩 관련된 것들을 자랑할 수 있는 자유 게시판이 있습니다.<br/>
상세 페이지로 이동하면 내가 쓴 글들을 페이지네이션으로 볼 수 있습니다.<br/>
chatGPT를 사용하여 “코린이들이 제일 힘들어하는!!! 변수명 이름 짓기”를 하여 이것에 대한 의견을 내며 자유로운 소통을 할 수 있습니다.<br/>
이 웹 페이지를 통해서 코린이들끼리 서로 정보를 공유하고 도와가며 많은 정보를 알아가 취업에 성공하는 지름길이 되었으면 좋겠습니다.<br/>
