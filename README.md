## 1. 팀 과제 소개
학습에 관한 질문을 할 수 있는 학습 게시판, 장비나 코딩 관련된 것들을 자랑할 수 있는 자유 게시판이 있습니다.<br/>
nav바에서 자유게시판을 누르면 자유게시판에 쓴 글만 보아서 볼 수 있습니다.<br/>								       
게시판의 글을 누르면 상세페이지로 들어가게 되고, 댓글도 작성할 수 있습니다.<br/>
마이페이지로 이동하면 내가 쓴 글들을 모아볼 수 있고 닉네임, 비밀번호를 변경 할 수 있습니다.<br/>
그리고 오른쪽 아래 버튼을 누르면 chatGPT를 사용하여 “코린이들이 제일 힘들어하는!!! 변수명 이름 짓기”를 하여 이것에 대한 의견을 내며 자유로운 소통을 할 수 있습니다.<br/>
이 웹 페이지를 통해서 코린이들끼리 서로 정보를 공유하고 도와가며 많은 정보를 알아가 취업에 성공하는 지름길이 되었으면 좋겠습니다.<br/>
